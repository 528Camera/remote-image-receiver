# remote-image-receiver
___

Сервис для хранения, анализа и обработки сообщений, принимаемых по ZMQ-сокету.
___

# Requirements
___

Требуемые к установке пакеты (Ubuntu):

* protobuf-compiler
* cmake, make, gcc, g++
* build-essential
* libzmq3-dev
* libjsoncpp-dev
* libboost-all-dev
* libcppunit-dev
* libopencv-dev
